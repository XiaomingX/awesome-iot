# awesome-iot
## Framework (框架)

### 常用框架

- **[ESP-IDF ★ 2542](https://github.com/espressif/esp-idf)**  
  Espressif官方开发框架，用于ESP32设备的Wi-Fi、BLE和蓝牙应用开发。
  
- **[Johnny-Five ★ 6024](https://github.com/rwaldron/johnny-five)**  
  基于JavaScript的物联网和机器人编程框架，使用Firmata协议支持多种硬件。

- **[GoBot ★ 2062](https://github.com/hybridgroup/gobot)**  
  用于机器人、物联网开发的Golang框架，支持多种硬件设备。

- **[Serverless ★ 7951](https://github.com/serverless/serverless)**  
  适用于Amazon Web Services Lambda和API Gateway的无服务器应用程序框架。

- **[Sming ★ 1800](https://github.com/SmingHub/Sming)**  
  高性能异步C/C++框架，专为嵌入式设备设计，支持多种网络功能。

- **[Freedomotic ★ 208](https://github.com/freedomotic/freedomotic)**  
  开源物联网框架，用于构建和管理智能空间。

## Library (库)

### SDK

- **[ESP8266 Arduino Core ★ 2588](https://github.com/esp8266/Arduino)**  
  适用于ESP8266 Wi-Fi芯片的Arduino开发核心库。

- **[Azure IoT SDK ★ 203](https://github.com/Azure/azure-iot-sdks)**  
  Microsoft Azure物联网SDK，支持多种语言和平台连接设备。

### Arduino相关库

- **[ArduinoJson ★ 873](https://github.com/bblanchon/ArduinoJson)**  
  用于嵌入式系统的高效JSON库。

- **[WiringPi ★ 455](https://github.com/WiringPi/WiringPi)**  
  类似于Arduino的Raspberry Pi GPIO控制库。

### 底层开发库

- **[Simbody ★ 540](https://github.com/simbody/simbody)**  
  用于多体动力学/物理仿真的高性能C++库。

- **[TinyOS ★ 543](https://github.com/tinyos/tinyos-main)**  
  专为低功耗无线设备设计的操作系统，如传感器网络和智能建筑应用。

## App (应用)

- **[MQTTX ★ 12](https://github.com/emqx/MQTTX)**  
  跨平台的MQTT桌面客户端，支持快速测试MQTT连接和消息发布/订阅。

- **[Theengs App](https://app.theengs.io)**  
  移动BLE到MQTT桥接应用，兼容90多种蓝牙传感器。

## OS (操作系统)

- **[Amazon FreeRTOS ★ 842](https://github.com/aws/amazon-freertos)**  
  专为微控制器设计的操作系统，便于开发、部署和管理低功耗边缘设备。

- **[Contiki ★ 1813](https://github.com/contiki-os/contiki)**  
  针对物联网的开源操作系统，适合智能建筑、传感器网络等应用。

- **[RIOT ★ 748](https://github.com/RIOT-OS/RIOT)**  
  友好的物联网操作系统，支持多种嵌入式设备。

- **[Zephyr ★ 352](https://github.com/zephyrproject-rtos/zephyr)**  
  小型实时操作系统，支持多种体系结构，适合资源受限设备。

## Security (安全)

- **[IoTSeeker](https://github.com/rapid7/IoTSeeker)**  
  扫描网络中物联网设备，检测是否使用默认工厂设置密码。

- **[nShield ★ 66](https://github.com/fnzv/nShield)**  
  基于iptables的轻量级Anti-DDoS解决方案，适用于物联网设备。
